Operating System

Operating system experiments:

  • Revision practice of various commands like man, cp, mv, ln, rm, unlink, mkdir, rmdir, etc., and many more that were learned in the IT Workshop course and later.
  • Implement two-way process communication using pipes.
  • Implement message queue form of IPC.
  • Implement shared memory and semaphore form of IPC.
  • Simulate the CPU scheduling algorithms - Round Robin, SJF, FCFS, priority.
  • Simulate Bankers' algorithm for Deadlock Avoidance and Prevention.
  • Simulate all FIFO Page Replacement Algorithm using C program.
  • Simulate all LRU Page Replacement Algorithms using C program.
  • Simulate Paging Technique of Memory Management.
  • Practice various commands/utilities such as cat, nl, uniq, tee, pg, comm, cmp, diff, tr, tar, cpio, mount, umount, find, umask, ulimit, sort, grep, egrep, fgrep, cut, paste, join, du, df, ps, who, etc., and many more.

DataBase Lab

DBMS experiments:

  • Case Study-1: Employee database – ‘Create’ employee table, ‘Select’ and display an employee matching a given condition, ‘Delete’ duplicate records, delete rows using triggers, insert and update records, find net salary, etc.
  • Case Study-2: Visitor Management database.
  • Case Study-3: Students Academic database.
  • Case Study-4: Inventory Management System database.
  • Case Study-5: Bank Operations database.
  • Case Study-6: Bus Operator (Roadways) – Prepare E-R Model, Relational Model, perform Normalization, Create Tables, Insert data, Delete Data, Query database, create stored procedures, etc.

Networking Lab

Experiments:

  • Showing various types of networking cables and connectors, identifying them clearly.
  • Looking at specifications of cables and connectors from various companies on the internet, identifying differences.
  • Making patch cords using different types of cables and connectors—crimping, splicing, etc.
  • Demonstration of different types of cable testers, using them for testing patch cords prepared by students in the lab and standard cables prepared by professionals.
  • Configuring computing devices (PC, Laptop, Mobile, etc.) for network, exploring different options and their impact—IP address, gateway, DNS, security options, etc.
  • Showing various networking devices—NICs, Hub, Switch, Router, WiFi access point, etc.
  • Looking at specifications of various networking devices from different companies on the internet, identifying differences.
  • Using a network simulation tool (e.g., Cisco Packet Tracer).
  • Setting up a small wired LAN in the lab.
  • Setting up a small wireless LAN in the lab.

Web technologies Lab

Experiments:

  • Coding Server Client Programs.
  • Developing Web Application using HTML, JavaScript.
  • Developing Advanced Web Application Programs using CSS.
  • Practicing PHP: Basics.
  • Practicing PHP: Web Application Development.
  • Practicing PHP: MySQL - tiered Applications.
  • Developing a fully functional Web Service Application using all the technologies learned in this course.
